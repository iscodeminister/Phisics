      ```ball.pos.x = -floor.length/2
	  ball.pos.y = ball.radius+floor.height/2
	  ball.y0 = ball.pos.y
	  ball.m = 0.1 
	  ball.v0 = 10
      ``` 
      運算完之後以以圖表確認並輸出csv紀錄實驗數據<br><br>
